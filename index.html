Semântica em HTML

(Voltar)

Semântica em HTML é: Estruturar o conteúdo de uma página de forma lógica e significativa, utilizando tags que descrevem o conteúdo, facilitando a navegação e a acessibilidade do site.

(Voltar)

Clients é: Quem inicia a interação com o servidor, ou seja, quem solicita a informação, enquanto o servidor é quem fornece a informação solicitada.

Como é uma estrutura básica de um HTML?

(Voltar)

A Estrutura básica de um HTML é composta por quatro elementos principais:

!DOCTYPE html, html, head e body.

O !DOCTYPE html, informa que estamos usando o HTML5, ou seja, estamos usando a versão mais recente do HTML.
O html: é o elemento principal que contém todos os outros elementos da página.
O head: é onde ficam as informações sobre a página, como o título, metadados, etc.
O body: é onde ficam as informações visíveis da página, como texto, imagens, etc.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Lista ordenada: É uma lista que segue um padrão de ordem numerica, como 1, 2, 3 e assim por diante.

Lista não ordenada: É uma lista que não segue nenhum padrão especifico, contendo apenas pontos ao lado de cada item.

Links de referências

(Voltar)

Links de referências são: Urls usadas para redirecionar o usuário para outra página ou site. Ao clicar em um link de referência, o leitor pode se aprofundar no tema, acessando artigos relacionados, livros e outros materiais relevantes para a pesquisa.
